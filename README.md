### Beef Burrito

Collaboration project between myself, Dark Koopatrol, Igrmi, and yoshyegg. Our goal is to introduce RNG Manipulation into Super Paper Mario speedruns - this project is currently focused on manipulating RNG in Chapter 1-1.

Much of this code (particularly spm-system.c) was taken directly from SeekyCt's Super Paper Mario system.c decompilation @ [spm-decomp](https://github.com/SeekyCt/spm-decomp).
